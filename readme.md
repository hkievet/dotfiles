## NERDTree tips

Use the bookmark capabilities on
* common project directories.
* dotfiles folder

* use `Bookmark <name of bookmark to set>` to add a bookmark where the cursor is
* use `OpenBookmark <name of bookmark>` to jump to that bookmark
